github添加了ssh key之后进行push还需要登陆的问题

Created2024-09-05|Updated2025-04-07|工具使用教程
|Post Views:

在github上添加了ssh key之后，使用git push推送代码还是需要登陆，并显示这种登陆方式已经移除，需要token

使用git remote -v查看，发现远程仓库的链接还是https形式，需要将其修改为ssh形式

使用git remote set-url origin git@github.com:xxxx.git修改远程仓库的地址

使用git remote -v查看发现远程仓库的地址已被修改，并且上传正常
